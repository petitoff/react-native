```js
class App extends React.Component {
  render() {
    return <View></View>;
  }
}

export default App;
```

Props have PropTypes

```js
static propTypes = {
  randomNumberCount: PropTypes.number.isRequired,
  initialSeconds: PropTypes.number.isRequired,
  onPlayAgain: PropTypes.func.isRequired,
};
```
